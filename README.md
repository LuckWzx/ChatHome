项目简介

GoChat 是一个基于 Go 语言实现的轻量级实时聊天室应用，采用 TCP 协议进行通信，具有高性能、低延迟的特点。项目专注于展示 Go 语言在并发编程方面的优势，特别优化了锁机制以避免常见的并发问题，适合学习 Go 语言网络编程和并发控制。
主要功能
多用户实时聊天
用户加入/离开通知
个性化用户名支持
简洁的命令行界面
并发安全的消息广播
用户名冲突检测
技术特点
高效的并发模型：使用 goroutine 处理每个客户端连接
精细的锁优化：避免嵌套锁导致的死锁问题，确保高并发下的稳定性
资源管理：自动清理断开连接的客户端，防止资源泄漏
清晰的代码结构：模块化设计，易于理解和扩展
错误处理：完善的错误处理机制，保证服务稳定性
快速开始

1. 克隆仓库:
bash
git clone [https://github.com/LuckWzx/ChatHome.git]
cd gochat

2. 启动服务器:
bash
go run server.go

3. 在新终端启动客户端:
bash
go run client.go

4. 按照提示输入用户名，开始聊天!
